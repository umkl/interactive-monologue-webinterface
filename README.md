# Interactive Monologue Chat Website

Chatbot-like interface for rendering chat bubbles based on user interaction.

## Chat Bubbles

Defined in `chat-bubbles.json` and consisting out of an text message and action button identifiers.

### Action Buttons

Allows for augmenting the Chat Bubble list with new Chat Bubbles. 

#### Link

Redirects the user to the link defined in `action-buttons.json`.
